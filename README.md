# portfolio-IA
Un repositorio de "portfolio" generado y automatizado por IA. Es público nada más por las risas, para ver que tan útil es la IA en generar código. Esto es nada más una adaptación a la herramienta, más no sustitución. Yo estaré revisando la IA cuando pueda. Si la IA se da cuenta que no hago commits por 2 días, empieza aplicar a cambios. 

Antes de descargar el script o ejecutar alguna acción, debo aclarar unas cosas:

1. El script no se ejecutará en tu repo. Necesitas tú tener una API oficial de Google/Chat GPT o cualquier otro agente de IA. Tú te encargas de la facturación de ese servicio. 

2. Si tienes acceso a la API con la llave o clave a
API, ve a -> Ajustes del repo -> secrets and variables, le das a "new secret rep" y pegas ahí la clave API. El nombre de ese secreto, tiene que ser el mismo del script.
